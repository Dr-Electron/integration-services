In this example, you will learn to authenticate your identity. Make sure to read the
[general authentication concept](../../explanations/authentication/concept.md), so you can fully understand the
[Authentication Workflow](../../explanations/authentication/concept.md#authentication-workflow).

```js
    secretKey: '5N3SxG4UzVDpNe4LyDoZyb6bSgE9tk3pE2XP5znXo5bF'
}
```
