# Create Identity and Credentials

The first example [example-1](https://github.com/iotaledger/integration-services/blob/develop/clients/node/examples/1-CreateIdentityAndCredential.ts)
performs the following tasks:

- Create a client for Identity features using API_URL and API_KEY environment variables
- Authenticate the client using the Admin identity (see [How to run examples](./how-to-run-examples))
- 


You can create an identity:

```js

import { ClientConfig, IdentityClient } from 'iota-is-sdk';

const config: ClientConfig = {
    apiKey: process.env.API_KEY,
    baseUrl: process.env.API_URL,
    apiVersion: ApiVersion.v01
};

const identity = new IdentityClient(config);

const userIdentity = await identity.create('Name Surname');
```
